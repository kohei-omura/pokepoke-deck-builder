# 🃏 ポケポケ デッキビルダー

**Pokémon TCG Pocket** のデッキレシピ・回し方・強み/弱みを一覧・検索できるWebアプリです。  
Grok AI によるプロトタイプを基に、Streamlit + JSON外部化で構築しました。

---

## 📸 概要

| 機能 | 説明 |
|------|------|
| デッキ一覧表示 | 収録デッキをカード形式で一覧表示 |
| デッキ詳細 | レシピ・回し方（序盤/中盤/終盤）・強み/弱み・対策を表示 |
| タイプ絞り込み | 炎/水/草/雷/超などタイプでフィルター |
| 新規デッキ追加 | UIからデッキを追加してJSONに保存 |
| デッキ削除 | 不要なデッキを削除 |
| CLIモード | `python main.py` でターミナルからも使用可能 |

---

## 🚀 インストール・起動方法

### 前提条件
- Python 3.9 以上
- pip

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/<your-username>/pokepoke-deck-builder.git
cd pokepoke-deck-builder

# 依存パッケージをインストール
pip install -r requirements.txt
```

### Webアプリ起動

```bash
streamlit run app.py
```

ブラウザが自動で開きます（デフォルト: `http://localhost:8501`）

### CLIバージョン起動

```bash
python main.py
```

## 🗂️ デッキデータ（decks.json）の構造

デッキデータはすべて `decks.json` で管理しています。  
新しいデッキを追加したい場合は、以下の形式でJSONに追記するだけです。

```json
{
  "デッキ名": {
    "タイプ": "炎",
    "難易度": "★★★",
    "レシピ": {
      "ポケモン": [
        {"名前": "ポケモン名", "枚数": 2}
      ],
      "トレーナー": [
        {"名前": "カード名", "枚数": 2}
      ]
    },
    "回し方": {
      "序盤": "序盤の動き",
      "中盤": "中盤の動き",
      "終盤": "終盤の動き"
    },
    "強み": ["強みの説明"],
    "弱み": ["弱みの説明"],
    "対策": "対策・アドバイス",
    "イメージカラー": "#FF6B35"
  }
}
```

## 🔧 拡張案

- **デッキ評価機能**: ユーザーが星評価を投票できるシステム
- **勝率記録**: 対戦結果を記録して勝率を表示
- **カード検索**: ポケモン名・トレーナー名でデッキを逆引き検索
- **デッキ比較**: 2つのデッキを並べて比較表示
- **Streamlit Cloud デプロイ**: 無料でWebに公開可能
- **エクスポート機能**: デッキレシピをテキストやCSVで書き出し
- **対策デッキ提案**: 弱点タイプを自動判定して対策デッキを推薦

## 🌐 Streamlit Cloud で公開する方法

1. このリポジトリを GitHub に Push
2. [https://streamlit.io/cloud](https://streamlit.io/cloud) にアクセス
3. 「New app」→ リポジトリ選択 → `app.py` を指定
4. 「Deploy」ボタンをクリック → URL が発行されます（無料）
